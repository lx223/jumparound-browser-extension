## Description (16,000 char limit)

### English

---

Cmd+K to instantly find any tab. Like VS Code's Quick Open, but for your browser.

50+ tabs across multiple windows? Stop clicking. Press Cmd+K (Ctrl+K on Windows/Linux), type a few letters, hit Enter. You're there.

KEY FEATURES

• Fuzzy Search - Type "gh iss" to find "GitHub Issues", or "dmd" for "DocsMainDashboard" (camelCase matching)
• Recent History - Closed a tab? Your recent browsing history appears right alongside open tabs
• Smart Ranking - Most recently used tabs surface first
• All Windows - One unified, searchable list across every window
• 100% Keyboard-Driven - Arrows to navigate, Enter to switch, Escape to dismiss

HOW IT WORKS

Cmd+K → "local" → Enter (localhost). Cmd+K → "stack" → Enter (Stack Overflow). Cmd+K → "github" → Enter (your PR). Done.

PRIVACY

Zero data collection. Zero tracking. Runs 100% locally in your browser.

## Single Purpose

---

JumpAround is a keyboard-driven tab switcher that enables users to quickly find and switch between open browser tabs using fuzzy search. The extension's sole purpose is to provide fast tab navigation through a searchable overlay activated by a keyboard shortcut (Cmd+K/Ctrl+K).

---

## Permission Justifications

---

tabs
Required to read information about all open tabs (titles, URLs, favicons) in order to display them in the searchable tab list and enable switching between them.

history
Required to retrieve recent browsing history so users can quickly navigate back to recently visited pages that are no longer open in any tab, displayed alongside open tabs in the search results.
